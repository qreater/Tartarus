title: "ConfigDefinitionAPI"
---

Class for CRUDL of Configuration Definitions.

#### Initialization

```py
from tartarus_lib import TartarusAPIClient

tartarus = TartarusAPIClient(
    base_url="<BASE_URL>",
    api_key="<API_KEY>"
)

cd_client = tartarus.config_definitions
```

#### Methods

<Accordion icon="c" title="reate Config Definition">

Create a new configuration definition.

```py
cd_create = cd_client.create(
    config_definition_key="test_config_definition",
    json_schema= <SCHEMA>,
    indexes= <INDEXES>
)
```

<Icon icon="scroll" /> **Parameters**:  
<ParamField  body="config_definition_key" type="string" required>
   The key for the config definition.
 </ParamField>
 <ParamField  body="json_schema" type="JSON" default="None">
   The JSON schema for the config definition.
 </ParamField>
 <ParamField  body="indexes" type="JSON" default="None">
   The list of indexes for the config definition.
 </ParamField>      

<Icon icon="arrow-turn-down-left" /> **Returns**: Success response with dictionary of the config-def data.


</Accordion>
